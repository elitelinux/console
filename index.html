How I can run these scripts in the console?

Make Exe $ chmod +x /path/to/scripts/mi_script.sh $ chmod 0755 /path/to/scripts/mi_script.sh

get into path

  • $ cd /path/to/scripts

then

  • $ ./mi_script.sh

or

  • $ ./path/to/scripts/mi_script.sh

or

  • $ sh mi_script.sh

or

  • $ source ./mi_script.sh

or

put in the PATH If you wanna ;

  • $ export PATH=$PATH:/path/to/scripts/

then

runs the script the way you like ;)
